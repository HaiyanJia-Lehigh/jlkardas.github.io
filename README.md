## Lehigh Enrollment Spring 2020
This pie chart is created using Datawrapper as a course project of COMM 165, Data Storytelling. It shows the undergraduate enrollment in the different colleges and programs at Lehigh University for Spring 2020. The chart shows that Engineering & Applied Science has the largest enrollment, followed by College of Business & Economics.
